# dotfiles

My personal Hyprland dotfiles on Debian, built on top of [JaKooLit's Debian-Hyprland](https://github.com/JaKooLit/Debian-Hyprland) installer.

## System

- **OS:** Debian (sid/testing)
- **WM:** Hyprland (Wayland)
- **Shell:** Zsh + oh-my-zsh
- **Monitors:** Acer XV275K P3 (4K 160Hz) + Lenovo L24q-30 (1440p, rotated portrait)
- **Installer:** [JaKooLit/Debian-Hyprland](https://github.com/JaKooLit/Debian-Hyprland)

## Hyprland Config Structure

The JaKooLit installer separates configs into two layers — **default configs** (updated by the installer) and **user configs** (yours to edit, never touched by updates).

```
config/hypr/
├── hyprland.conf          # Main entry point — sources everything below
├── monitors.conf          # Monitor layout (generated by nwg-displays)
├── workspaces.conf        # Workspace-to-monitor assignments
├── hypridle.conf          # Idle / screen timeout settings
├── hyprlock.conf          # Lock screen config
├── hyprlock-1080p.conf    # Lock screen variant for 1080p
├── hyprlock-2k.conf       # Lock screen variant for 1440p
│
├── configs/               # DEFAULT configs — from the installer
│   ├── Keybinds.conf      # Built-in keybinds (don't edit; override in UserConfigs)
│   ├── SystemSettings.conf
│   ├── ENVariables.conf
│   ├── WindowRules.conf
│   ├── Startup_Apps.conf
│   └── Laptops.conf
│
├── UserConfigs/           # YOUR configs — safe to edit, not overwritten by updates
│   ├── 00-Readme          # Explanation from installer
│   ├── 01-UserDefaults.conf   # Default apps: $term, $files, $EDITOR, search engine
│   ├── UserSettings.conf      # Main Hyprland settings overrides
│   ├── UserKeybinds.conf      # Add/override keybinds here
│   ├── UserDecorations.conf   # Window decorations / blur / shadows
│   ├── UserAnimations.conf    # Animation config
│   ├── ENVariables.conf       # Your env variable overrides
│   ├── WindowRules.conf       # Your window rules
│   ├── Startup_Apps.conf      # Apps to autostart
│   ├── Laptops.conf           # Laptop-specific settings
│   └── LaptopDisplay.conf     # Laptop display config
│
├── UserScripts/           # YOUR scripts — safe to edit
│   ├── WallpaperSelect.sh     # Wallpaper picker
│   ├── WallpaperRandom.sh     # Random wallpaper
│   ├── WallpaperAutoChange.sh # Auto wallpaper rotation
│   ├── WallpaperEffects.sh    # Wallpaper shader effects
│   ├── RofiBeats.sh           # Music via Rofi
│   ├── Weather.sh / Weather.py # Weather widget
│   └── ...
│
├── scripts/               # INSTALLER scripts — from JaKooLit (don't edit)
│   ├── *.sh               # Hyprland utility scripts (brightness, volume, etc.)
│   └── keybinds_parser.py
│       # NOTE: dots-tui binary excluded from this repo (17MB compiled binary)
│       # Get it from the Debian-Hyprland installer
│
├── animations/            # Animation presets (selectable via Rofi)
├── wallust/               # Wallust color scheme templates for Hyprland
└── Monitor_Profiles/      # Saved monitor layout profiles
```

### Key Files to Customize

| File | What to change |
|---|---|
| `UserConfigs/01-UserDefaults.conf` | Default terminal, file manager, editor, search engine |
| `UserConfigs/UserKeybinds.conf` | Add your own keybinds (unbind first if replacing) |
| `UserConfigs/Startup_Apps.conf` | Apps to launch on login |
| `UserConfigs/ENVariables.conf` | Nvidia, scaling, and other env vars |
| `monitors.conf` | Monitor layout — best managed via `nwg-displays` |

## Waybar

```
config/waybar/
├── config → configs/TOP-Default   # Active layout (symlink)
├── style.css → style/Extra-Prismatic-Glow.css  # Active style (symlink)
├── configs/        # Many layout presets (TOP, BOT, LEFT, RIGHT, combined)
├── style/          # Many CSS theme files (Catppuccin, Wallust, Aurora, etc.)
├── Modules/        # Module definitions used by layouts
├── ModulesCustom/  # Your custom modules
├── ModulesGroups/  # Grouped module definitions
├── ModulesVertical/ # Modules for vertical bars
├── ModulesWorkspaces/ # Workspace-specific modules
├── UserModules/    # User-added modules
└── wallust/        # Wallust color templates for Waybar
```

Switch layouts/styles interactively via **SUPER+SHIFT+W** (Waybar style picker via Rofi).

## Color Theming (Wallust)

[Wallust](https://codeberg.org/explosion-mental/wallust) generates color schemes from wallpapers and applies them across the desktop.

Templates live in each app's `wallust/` subdirectory:
- `config/hypr/wallust/` — Hyprland colors
- `config/waybar/wallust/` — Waybar colors
- `config/rofi/wallust/` — Rofi colors
- `config/wallust/` — Wallust config and templates

Wallpaper + theming is changed via **SUPER+W** (wallpaper picker) or the WallpaperSelect/WallpaperRandom scripts.

## Installing on a Fresh System

> **Prerequisites:** Run [JaKooLit's Debian-Hyprland installer](https://github.com/JaKooLit/Debian-Hyprland) first to get all dependencies, then overlay these dotfiles.

```bash
git clone https://github.com/najchris11/dotfiles.git ~/dotfiles
cd ~/dotfiles
bash install.sh
```

The install script backs up any existing configs and symlinks everything into place.
